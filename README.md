## Quick Start

> 👉 Before run this script make sure that you have already created a Python 3 virtual environment ;)

```sh
# Execute "run.sh" script passing as argument the name of scprit file.
sh run.sh $SCRIPT

# Example
sh run.sh src/first.py
```
